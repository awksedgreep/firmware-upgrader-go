üîå API Overview

The Firmware Upgrader REST API provides programmatic access to all system functionality. All endpoints return JSON responses and use standard HTTP methods.

Base URL

http://your-server:8080/api

Response Format

All API responses follow this structure:

{ "success": true, "data": { ... }, "error": null }

Error Handling

Errors return appropriate HTTP status codes with details:

{ "success": false, "data": null, "error": "Error message here" }
